# Umair's Fast Food Ordering System 🍔🍕🍝

This is a simple Python project that simulates a fast-food ordering system using dictionaries.

## Features
- Displays a food menu
- Allows customers to order one or more items
- Calculates total bill

## How to Run
1. Install Python (3.8 or above)
2. Clone this repo:
3. python Order_Food.py
